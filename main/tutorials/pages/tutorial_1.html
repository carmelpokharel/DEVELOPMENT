Introduction

What is programming?

In a nutshell, programming refers to the process of making a computer do something for you. This involves three basic steps:

  1. Providing an input to the computer
  2. Computer processing your instructions
  3. Receiving an output from the computer

What is a programming language?

To communicate with computers, we use languages, much like the ones we use amongst ourselves everyday. In the context of programming, a language is a set of rules which govern what we may input to a computer.

There are many programming languages to choose from, and you may have even heard of a few, such as C++, Java, Python, or Ruby. Each language comes with its own merits and shortcomings, but they all fundamentally do one thing: provide a means to communicate with your computer.

Why Python?

When choosing your first programming language, you'll want to pick something that's marketable, simple, powerful, friendly, and well-supported.

While some of these qualities may seem almost contradictory, Python perfectly fits the bill.

Whether you're working with web applications, desktop programs, quick user scripts, automation tools, lesson plans, prototypes, or scientific number crunching, Python does it all.
