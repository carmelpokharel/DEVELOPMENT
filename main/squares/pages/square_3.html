Who's Afraid of 138?

Now for the int datatype, which represents integers in Python.

If you recall from math class, an integer is any whole number that is positive, negative, or zero. Python's int datatype is much the same.

Printing an int is almost like printing a str, but with one difference: unlike strings, integers don't need quotation marks.

Task Three

Print the integers 138 and -138 on separate lines, so that the output looks like the following.

138

-138
