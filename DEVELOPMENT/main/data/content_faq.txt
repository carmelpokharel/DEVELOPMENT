Q: Which version of Python is used in Pythons and Ladders?
<p>Pythons and Ladders makes use of the Python emulator from <a href="http://www.skulpt.org" target="_blank">Skulpt.org</a>, whose currently known supported version is Python 2 (2016).
Q: How can I install and set up Python on my computer?
<p>Although this step isn't a requirement for Pythons and Ladders, <a href="https://wiki.python.org/moin/BeginnersGuide/Download" target="_blank">here's a link</a> which guides you through the setup process. As far as editors go, we recommend <a href="http://www.wingware.com/downloads/wingide-101" target="_blank">Wing IDE 101</a> if you've never coded before, and in all other cases, <a href="https://www.sublimetext.com/3" target="_blank">Sublime Text</a>.
Q: When can I expect to complete Pythons and Ladders?
<p>For a beginner, the completion time is intended to be between a few weeks to a few months, depending on interest and engagement levels.
Q: What will I learn from Pythons and Ladders?
<p>Pythons and Ladders covers roughly the same amount of material as the first four weeks of a first-year university course, with a heavier focus on practical applications.
Q: How does Pythons and Ladders differ from other e-Learning tools?
<p>From a functional perspective, we developed Pythons and Ladders to mitigate the following issues in alternate services:<ul><li class="normal">Lack of focus on problem-solving methods & search engine querying skills<li class="normal">Exercises have no practical or real-world value (e.g. multiple-choice questions tracing code, inability to test functions, discouragement of open-ended solutions, etc.)<li class="normal">Neglect for mobile support and optimizations for slower connections<li class="normal">Quality content available only for a fee/subscription, with the "free" or "demo" packages offering inconclusive material</ul>
