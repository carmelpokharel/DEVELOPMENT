Form Follows Function

You've encountered some built-in functions so far, like len(), range(), and type(). In Python, you're also able to define your own functions, using the def keyword.

Here's a simple example to get started:

def exampleFunction():

    return 'Testing, testing, 123'

Notice the use of the new keyword return here, which returns a value, but does not display it to the Output box. In order to see the results of exampleFunction(), you can call it with a print statement outside of the function body:

def exampleFunction():

    return 'Testing, testing, 123'

print exampleFunction()

Task Forty-Two

Define a function, myFirstFunction(), and have it return 'Creating a function is easy!'. Then, print your function from outside the function body.
