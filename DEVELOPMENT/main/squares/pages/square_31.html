From the Inside

Can you access a substring from a str inside a list?

letters = ['ABCD','EFGH','IJKL']

print letters[1]

print letters[1][1]

In the above example, the first set of [square brackets] accesses the list item 'EFGH'. Then, the second set of square brackets returns the substring of 'EFGH', which is 'F'.
