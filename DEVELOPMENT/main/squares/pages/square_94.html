Check Yes, Juliet

A Shidoku is a Japanese number puzzle, related to the Sudoku (if you are unfamiliar with Sudokus, read about them here). In a properly formed 4 × 4 Shidoku, each row, each column and each interior 2 × 2 box contains the integers 1-4 exactly once.

Below is an example of a well-formed, completed 4 × 4 Shidoku:

1 3 2 4
4 2 1 3
3 1 4 2
2 4 3 1

In Python, you can represent this as:

[[1, 3, 2, 4], [4, 2, 1, 3], [3, 1, 4, 2], [2, 4, 3, 1]]

Task Ninety-Four

Define a function, checkShidoku(), which takes in a list representing a 4 × 4 Shidoku, and determines whether it is properly formed. Your output should be a bool value.
