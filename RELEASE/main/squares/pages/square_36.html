Vowels

Handily, for loops can also be used with if/else statements:

numbers = [0, 1, 2, 0, 3, -5, 7]

for digit in numbers:

    if digit > 0:

        print digit, 'is positive'

    else:

        print digit

Task Thirty-Six

Line 1Declare a variable, vowels_list, for the str 'AEIOU'
Line 2Declare a list variable, letters_list, for the str 'ABCDEF'
Lines 3-7Write a for loop to iterate through letters_list, and print each of its elements followed by ' is a vowel!' or ' is not a vowel.'

Your output should look like:

A is a vowel!

B is not a vowel.

C is not a vowel.

D is not a vowel.

E is a vowel!

F is not a vowel.
