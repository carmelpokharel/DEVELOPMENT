Cities of Night

Run the following code from your Editor, and observe the output:

cities = ['Toronto', 'Singapore']

cities = ['Tokyo'] + cities

print cities

Task Twenty-Nine (Easy)

Initialize the cities variable exactly as above. Then, add the list ['Tokyo', 'Seoul'] to the front of cities, and return it so that your output is exactly the following:

[['Tokyo','Seoul'], 'Toronto', 'Singapore']
