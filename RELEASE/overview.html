AUDIENCE

Our intended users are individuals interested in learning how to code, with the ability to motivate and pace themselves. More advanced users will likely not find this resource useful.

PREREQUISITES

In general, the questions in Pythons and Ladders assume the user has a basic knowledge of elementary algebra and mathematics.

USAGE

Powered by the Python emulator from Skulpt.org, Pythons and Ladders guides you through a sequence of squares, each containing a task. Your task is to complete each square, each more difficult than the preceding one, until you reach the end. Along the way, you will encounter some "Python" squares, and some "Ladder" squares, representing easier and harder versions of your current task. Use these alternative versions to help consolidate your learning.

 

RESOURCES

One of the most important skills you can develop as a programmer is your ability to ask and query the right questions.

We find the following resources especially useful:

StackOverflow.com — Every Python question you'll ever think of has been asked and answered here.

Python web documentation — Although many things will fly over your head here, try to study the usage-case examples.

PEP 008: Style guide in Python — Provides guidelines on how to keep your code clean, simple, and quintessentially "Python."

PEP 20: The Zen of Python — An Animal Farm-esque list of principles for Pythonic coding.
