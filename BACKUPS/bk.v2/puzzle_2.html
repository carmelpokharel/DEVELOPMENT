Variables are an important concept in programming. Think of them as boxes. They

are used to store things, and when you want, you can access their contents. You can

also carry them around without ever having to see or touch what's inside.

 

When naming your variables, try to make them unique. For example, a terrible variable

name would be something like "print". Why? Well "print" just happens to be a keyword

in Python. If you tried to execute "print = 5", Python is confused. What are you

trying to do? Luckily, since Python makes the final decision, it will refuse to run.

A great step is to avoid using any word in the English language as a variable name.

So, instead of "variable = 4", try "var1 = 4" instead.

 

Now, write Python code that stores the string "Pythons" in a variable called var1,

"and" in a variable called var2, and "Ladders!" in a variable called var3.
