Puzzle 4


SQ. 4

 

Now that you've learned how to add integers, how can you use Python to add non-integer

numbers? Hint: it's a new data type.

 

Extend your solution to Sq. 3 so that you are now able to handle any numerical data-

type in Python, and print it correctly to the output. Instead of addition, however,

you will now be performing multiplication.

 

For example, if you store the numbers 4.8 and 9 in num1 and num2 respectively, your

output should read exactly as follows:

▸ The product of 2.5 and 3 is 7.5.

 

Note: The following output is considered incorrect:

▸ The product of 2.5 and 3.0 is 7.5.

 

Approximate learning time

10-15 minutes


Questions for you

  1. What purpose do the hashtags on the first few lines serve?

  2. Which data type allows you to represent decimal numbers in Python?

  3. Once you have discovered this new data type, can you verify it by using isinstance() on num1?

  4. Can you convert between numerical data types? If so, how?

  5. What happens if you try to print int(4.8)?

  6. Can you convert between a string and this new data type?

  7. How many decimal points can this new data type store?

  8. Examine the test cases carefully. How does each pair test your code?
