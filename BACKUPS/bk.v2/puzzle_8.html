SQ. 8

 

To access an element in a list, we use a concept called indexes. Each element

in a Python list is assigned an index. These indices start at 0, and increment

with each subsequent element.

 

In the list ['2', '4', '3', '1', '5'], the element at index 4 is '5'.

 

To grab the element at a particular index, we use square brackets at the end

of the list, and include the index we'd like to read (see examples in the

editor). We can also grab multiple elements from a list at once, and determine

its length using the built-in Python function len().

 

Write a single-line statement that prints the last element of a givenlist. The

list may be of any length. If the list is empty, return the string 'N/A'. Test your code

on the following cases:

[]

[1,4,6,1,5,6]

should output exactly the following:
