Puzzle 3


SQ. 3

 

Write a Python script that displays the result of adding two numbers after calculating

their sum. Store your numbers in the variables num1 and num2, calculate their sum, and

print the result to the output.

 

For example, if you store the numbers 4 and 9 in num1 and num2 respectively, your

output should read exactly as follows:

▸ The sum of 4 and 9 is 13.

 

Approximate learning time

15-20 minutes


Questions for you

  1. How can you print variables storing both numbers and strings on the same line?

  2. Can you convert back and forth between string and integer types? How would you do this?

  3. What are the differences between these statements?
    print '7', '7'
    print '7' + '7'
    print 7, 7
    print 7 + 7

  4. What happens when you try multiplication with strings?
    print '7' * 7
    print 7 * 7
