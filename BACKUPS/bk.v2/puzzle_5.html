Puzzle 5


SQ. 5

 

A Boolean is a data type which is either True or False. Note that a Boolean is not a

string of the words "True" and "False", but its own data type.

 

To obtain Booleans, we can perform comparison operations, such as greater than, less

than, or equal to.

 

Run the sample code embedded in the editor, and analyze the output at the bottom.

Now, write 3 print statements which print the comparisons of two numbers.

 

For example, if your first number is 1.0, and your second number is 1, your statements

should output exactly the following 3 separate lines:

▸ Equality comparison: True

▸ Less than comparison: False

▸ Greater than comparison: False

 

Approximate learning time

7-10 minutes
