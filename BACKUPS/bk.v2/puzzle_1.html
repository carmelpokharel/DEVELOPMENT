Puzzle 1


SQ. 1

 

Run the sample code embedded in the editor, and analyze the output at the bottom.

 

Now, write a single-line print statement that outputs exactly the following line:

▸ "Teach me," she said, "I'm in for the long haul."

 

Note: Ignore the arrow; it's only there for you to distinguish the expected

output from the rest of the instructions.

 

Approximate learning time

10-15 minutes


Questions for you

  1. What happens to the quotations on the first two lines when you run the code?

  2. Why are there backslashes before some of the quotations?

  3. What did that '\n' do to "Quick lesson"?

  4. Does it matter if your words are enclosed in single quotes or double quotes?

  5. Why is there an empty line between '"I'm ready."'' and 'Quick lesson:'?
