Puzzle 6


SQ. 6

 

Here's the basic construct of an if/else statement:

 

if (conditional statement is True):

    do something

else:

    do something else

 

Recall from the previous lesson that a conditional statement always returns a

Boolean value: True, or False.

 

Run the sample code embedded in the editor, and analyze the output at the bottom.

Now, rewrite the if/else statement so that it does the exact same thing, but

uses "!=" as the comparison operator (and not "==").

 

Remember: we are naming our variable newSum, because we don't know if Python has

a built-in keyword or function named "sum"... we don't want any conflicts!

 

Approximate learning time

10-15 minutes


Questions for you

  1. What if you want to do something "if it's True", but do nothing "if it's False"?
    Can an if statement exist without an else statement?

  2. Can an else statement exist without an if statement?

  3. What if, in the editor's else statement, we changed "print firstNum + secondNum" to "newSum = firstNum + secondNum"? What would go wrong?

  4. In the editor's print statement, why does the following code not work?
    print "The new sum is " + newSum

  5. What's the difference between these two statements?
    print "The new sum is " + str(newSum)
    print "The new sum is ", str(newSum)
