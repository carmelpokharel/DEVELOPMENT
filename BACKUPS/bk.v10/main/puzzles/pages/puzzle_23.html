What If

Sometimes, you'll want to write code that runs only when a condition is met. An if statement helps you do just that. Here's the basic construct:

# if conditional statement is True:

#    do something

And here's an example of how you'd use an if statement:

final_grade = 80

if final_grade > 50:

    print 'You passed the course!'

Note: if statements must always end with a colon (:).

In the Python language, when the last character of a line is a colon (:), the next line must be indented. You can achieve the indentation with space characters.
