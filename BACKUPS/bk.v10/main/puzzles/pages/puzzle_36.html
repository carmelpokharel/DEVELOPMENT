Vowels

Handily, for loops can also be used with if/else statements:

numbers = [0, 1, 2, 0, 3, -5, 7]

for digit in numbers:

    if digit > 0:

        print digit, 'is positive'

    else:

        print digit

Task Thirty-Six

Line 1 Declare a variable, vowels_list, for the str 'aeiou'
Line 2 Declare a list variable, letters_list, which contains the first six letters of the English alphabet
Lines 3-7 Write a for loop that iterates through letters_list, and prints each of its elements followed by 'is a vowel!' or 'is not a vowel.'

Your output should look like:

A is a vowel!

B is not a vowel.

C is not a vowel.

D is not a vowel.

E is a vowel!

F is not a vowel.
