Interface

Welcome to Pythons and Ladders! First off, let's get you familiar with our interface.

INSTRUCTIONS

In this area, you'll find the instructions to complete your puzzles. Most of the time, it will also contain a quick lesson and provide you with expected results.

TOOLBAR

The set of buttons below is what we'll call the Toolbar. The Toolbar allows you to interact with the Editor and Output boxes (further down below). Currently, the available options are Run, Clear, Save, and Hint. Try them out!

RUN
CLEAR
SAVE
HINT

 

EDITOR

This first grey box is our Editor. You can click inside this area and write Python code. If necessary, you can also resize the Editor to see more of your code.

OUTPUT

This second grey box, below the Editor, is the Output. This box automatically expands to display the results of your Python code when you click "Run" in your toolbar.

PYTHON SHELL

        
The area below is an interactive Shell. Here, you can write some short code to see how Python interprets your commands. To use it, type in a statement and press Enter on your keyboard.

            
          
QUESTIONS FOR YOU

            
This section will contain a list of questions to help you understand your current puzzle in greater detail. Make sure to carefully examine them, as the results of some questions help in completing subsequent puzzles!

          
TEST CASES

            
Further along in your puzzles, you will be writing Python functions, and will need to test your functions with a variety of inputs. 

            

            
The test cases provided here will mainly help you to determine whether your solution is thorough.
