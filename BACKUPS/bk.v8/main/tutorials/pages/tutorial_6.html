Tutorial: Interface (II)

You may have noticed Clear or Save buttons next to Run on the previous two pages. In each Pythons and Ladders puzzle, there's one more: the Hint button.

These buttons represent what we'll call the Toolbar—a set of controls which allow you to manipulate the Editor and Output boxes.

Play around with the buttons below, and get a feel for how they work.
