Tutorial


Welcome to Pythons and Ladders! In this web-based game, you will be solving a series of consecutive puzzles using the Python programming language. The objective is to start with someone who has never worked with a single line of code, and leave them in a place where they understand some key programming concepts. The best part? You can play all of this in-browser, thanks to a fantastic team over at Skulpt. As a player, you begin with a game board at the 'INIT' square. After you solve the puzzle in the first square, you will be allowed to move on. The game is designed such that completing each puzzle enables you to attempt the next one. To solve the puzzles, you may use any resource available to you. This is for your own learning purposes, so use the problem sets to your advantage. For a quick example, consider the following puzzle below. You may enter any code and click "Run" as many times as you like to view the output, but you will only be assessed when you click "Submit." Submit compares your solution to my solutions, and if your code is sufficiently correct, you will unlock the next square. You may only submit 3 solutions per puzzle in a 24-hour period, so it'd be a good idea to test your answers before handing them in. Write a one-line Python statement which prints "Let's get started!" to the output. Note that incorrect code will not yield any output.
