My Favourite Things

A list is yet another data type in Python. A list in Python could be thought of as a shelf, carrying items. You can store just about anything on this shelf - integers, floats, strings, variables... even another list!

A Python list is enclosed in square brackets, with each element separated by a comma:

listName = [content0, content1, content2]

 

You can also check within a list to see if it contains a particular element. For example:

print content1 in listName    # would result in True

print content5 in listName    # would result in False
