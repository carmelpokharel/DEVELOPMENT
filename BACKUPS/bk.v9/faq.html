When can I expect to complete Pythons and Ladders?

For a complete beginner, the completion time is intended to be from one to three months. This, of course, is proportional to the activity and interest levels of the individual.

Which version of Python is used in Pythons and Ladders?

Pythons and Ladders makes use of the Python emulator from Skulpt.org, whose currently known supported version is Python 2.6+ (2016).

How can I install and set up Python on my computer?

Here's a link which guides you through the setup process. As far as editors go, we recommend Wing IDE 101 if you've never coded before, and in all other cases, Sublime Text.

How can Python be relevant to my everyday work and lifestyle?

We've attempted to cover real-world use cases in the last few squares of Pythons and Ladders, but here's an article which answers this question far better than we can.

Why is nothing happening when I run my code?

The absence of error messages in the output box is a current limitation in emulator technology. Please see this page of the Tutorial for more information, or install Python on your local machine.

Why can't I use the tab key to indent code in my Editor?

While we figure out a solution, try using spaces instead! When it comes to indentation, either 2 or 4 spaces are best. Whichever you pick, remember to stay consistent.
