Which version of Python is used in Pythons and Ladders?

Pythons and Ladders makes use of the Python emulator from Skulpt.org, whose currently known supported version is Python 2 (2016).

How can I install and set up Python on my computer?

Although this step isn't a requirement for Pythons and Ladders, here's a link which guides you through the setup process. As far as editors go, we recommend Wing IDE 101 if you've never coded before, and in all other cases, Sublime Text.

When can I expect to complete Pythons and Ladders?

For a beginner, the completion time is intended to be between a few weeks to a few months, depending on interest and engagement levels.

What will I learn from Pythons and Ladders?

Pythons and Ladders covers roughly the same amount of material as the first four weeks of a first-year university course, with a heavier focus on practical applications.

How does Pythons and Ladders differ from other e-Learning tools?

From a functional perspective, we developed Pythons and Ladders to mitigate the following issues in alternate services:

  • Lack of focus on problem-solving methods & search engine querying skills
  • Exercises have no practical or real-world value (e.g. multiple-choice questions tracing code, inability to test functions, discouragement of open-ended solutions, etc.)
  • Neglect for mobile support and optimizations for slower connections
  • Quality content available only for a fee/subscription, with the "free" or "demo" packages offering inconclusive material

 

RESOURCES

As developers, we understand that not everyone shares the same learning style.

In the interest of competition-free education, here are some similar, well-executed resources you may find useful:

Automate the Boring Stuff — In our opinion, the perfect sequel to Pythons and Ladders.

LearnPython.org — Uses a similar interactive & in-browser construct, and is great for reference.

SoloLearn — A more standardized approach to coding, complete with lessons and quizzes.

Codeacademy — A course-based offering, with plenty of options for learning other languages as well.

Code School — A game-themed approach with pricing plans for personal and business use.
